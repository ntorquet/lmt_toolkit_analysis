# LMT-toolki API documentation

## Django rest framework api 3.13.1 (api/v1)

## Every url start with api/v1

### Files
- **/files**
  - **GET /files** Return a file list
  - **POST /files**

### File
- **/files/<pk:int>**
  - **GET /files/<pk:int>** Return the file
  - **DELETE /files/<pk:int>** Delete the file
